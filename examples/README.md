# Examples

Examples live in each provider folder's main file.

- Start with `docs/quickstart.md`, then pick a task guide from `docs/README.md`.

- See `examples/*/` with `main.ts` or equivalent for DSL usage examples.
- `examples/provider-cache-pinning/` runs a real OpenAI A/B test (requires `OPENAI_API_KEY`) and asserts the pinned metadata wiring.
