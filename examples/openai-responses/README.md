# Cria + OpenAI Responses API Example

This example shows how to use Cria with the OpenAI Responses API for reasoning models.

## Setup

```bash
pnpm install
```

## Run

Assumes `OPENAI_API_KEY` is set in your environment.

```bash
pnpm start
```

## Notes

The Responses API is designed for reasoning models. The `Reasoning` component can be used
to replay reasoning from previous API responses when continuing conversations.
