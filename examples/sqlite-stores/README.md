# SQLite stores

Use SQLite KV store and sqlite-vec vector store together.

## Prerequisites

- `OPENAI_API_KEY`
- `SQLITE_VEC_PATH` (path to the sqlite-vec `vec0` extension)

Tip: in your shell, export the key before running:

```bash
export OPENAI_API_KEY=...
export SQLITE_VEC_PATH=...
```
## Run

```bash
pnpm install
pnpm start
```
